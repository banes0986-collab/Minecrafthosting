# 🛡️ LegacyNW Network - Yönetim Paneli
LegacyNW Minecraft sunucusu için özel olarak geliştirilmiş, güvenli yönetim ve market arayüzü.

## 🚀 Özellikler
- **Yönetici Paneli:** `triggerbabaa` hesabı ile tam kontrol.
- **Rank Sistemi:** VIP, VIP+, LVIP ve LVIP+ tanımlama arayüzü.
- **Ticket Sistemi:** Oyuncu destek taleplerini yönetme.
- **Bakiye Sistemi:** Discord entegreli bakiye yükleme yönlendirmesi.

## 🛠️ Kurulum
Bu depoyu indirin ve `index.html` dosyasını tarayıcıda açın. Admin paneli için `admin.html` dosyasını kullanın.
